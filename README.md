# micro-frontend-react
 


## How To Use

To clone and run this application, From your command line:

```bash
# Clone this repository

# Go into the repository
$ cd micro1

# Install dependencies
$ npm install
$ npm run build
$ cd dist
$ lg
$ npx servor
$ cd ..

# Run the app
$ npm start

# Go into the repository
$ cd micro2

# Install dependencies
$ npm install

# Run the app

```
